# Code Review Command

Perform a comprehensive code review of the specified file or recent changes:

1. **Code Quality**: Check for clean code principles, readability, and maintainability
2. **Bug Detection**: Identify potential bugs, edge cases, and logic errors
3. **Security**: Flag common security vulnerabilities (OWASP Top 10)
4. **Performance**: Note any performance concerns or optimization opportunities
5. **Best Practices**: Ensure adherence to language-specific conventions

Provide actionable feedback with specific line references and suggested fixes.
